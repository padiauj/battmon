# battmon
A simple battery monitoring tool for Linux operating systems

# Installation
## Ubuntu (Bionic, Xenial, Precise, Trusty, Cosmic) 
```
sudo add-apt-repository ppa:padiauj/battmon
sudo apt-get update
sudo apt-get install python3-battmon
```

## Manual 

### Dependencies 
* Python 3
* setuptools>=20.7.0
* matplotlib>=2.1.2
* numpy>=1.14.0
* PyQt4>=4.11.4

```
make 
sudo make install 
```
